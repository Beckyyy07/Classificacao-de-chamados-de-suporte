 <h1> Sistema de Classificação de Chamados de Suporte </h1>

**Objetivo:** Sistema que analisa mensagens enviadas por clientes em sistemas de suporte ou atendimento, utilizando técnicas de Processamento de Linguagem Natural (PLN) 
para classificar automaticamente cada solicitação em **categorias operacionais: Incident, Request, Problem, Change**
